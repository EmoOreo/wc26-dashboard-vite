# WC26 Dashboard

Version 0.5.0 adds dashboard polish, Groups, Venues, group detail pages, venue detail pages, countdown labels, and cleaner home match-center sections.

## Deploy

This repo includes a prebuilt `dist/` folder. GitHub Pages deploys that folder using GitHub Actions.

## Local build

```bash
npm install
npm run build
```
